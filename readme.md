# Miami Vice Calculator

A retro-styled calculator with a Miami Vice aesthetic, built with vanilla HTML, CSS, and JavaScript.

## Features

- Basic arithmetic operations (add, subtract, multiply, divide)
- Chained operations support
- Decimal number support
- Keyboard input support
- Backspace functionality
- Division by zero error handling
- Responsive design

## Live Demo

[View Calculator](https://pankratiast.github.io/miami-vice-calculator/)

## How to Use

### Mouse/Touch
Click the buttons to input numbers and operators.

### Keyboard
- **0-9**: Input digits
- **+ - * /**: Operators
- **Enter or =**: Calculate result
- **Backspace**: Delete last digit
- **Escape or Delete**: Clear all
